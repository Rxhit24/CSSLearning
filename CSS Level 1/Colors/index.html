This is Heading 3 with coloring using Hexadecimal (hex codes)

This is paragraph using hexadecimal Colors {#ff02d9}. first two place represents amount of red color, next 2 amount of green and last 2 amount of blue color. {hexadecimal is number system conating 16 characters from 0 - 9 and a - f} And there Range is 00 - ff , 0 indicating none and ff as maximum amount, for example #ff0000 stand for Red similarly#00ff00 for green and #0000ff for Blue

We can also change background color using background-color tag of CSS example {background-color: colorname;} or we can use hex code like this{background-color:#ffffff;}, here the range will be same as in previous section.
