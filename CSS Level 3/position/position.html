Position playground

position: static/relative/absolute/fixed/sticky

static -> default position (top,right,bottom,left and z-index properties have no effect)

relative -> element is relative to itself(top,right,bottom,left and z-index properties will work)

relative means its first position or default position

absolute -> positioned relative to its closest positioned ancestor (removed from flow)

Removed from flow means it removed from its original position and is floating on top of documnet like boat on water

fixed -> positioned relative to browser (removed from flow)

sticky -> positioned based on user scrolled position
