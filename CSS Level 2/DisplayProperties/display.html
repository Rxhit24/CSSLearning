display: inline/block/inline-block/none;

inline -> takes only the space required by the element (no margin/ padding) example- span, a,

block -> takes full space available in width i.e entire page width example- div, h1, p etc

inline-block -> similar to inline but allows margin, padding and height & width

none -> to remove element from doxument flow
