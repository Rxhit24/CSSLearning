"display: flex;" is used to convert a block elements in to a flexBox container

on flex container we can apply flex properties such as..,

1."flex-direction: row/row-reverse/column/column-reverse;"

   » it sets how flex items is placed in container, along which axis and direction.

2."justify-content: flex-start/fles-end/center/space-around/space-between/space-evenly;"

   » it align items along main axis

3."flex-wrap: wrap/nowrap/wrap-reverse;"

   » it sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.

4."align-items: flex-start/flex-end/center/end;"

   » alignment along cross axis

5."align-content: center/end/flext-start/flex-end/baseline;"

   » align items along cross axis if items are in multiple line

flex properties that are applied on flex container items that is inside container item

        » 1. align-self: center/flex-start/flex-end/baseline;

        » 2. flex-grow: (grow size compare to other items);

        » 3. flex-shrink: (shrink size compare to other items);
